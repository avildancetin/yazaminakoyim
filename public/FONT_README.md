# W95FA Font Setup

To use the W95FA font, place the font file in this `public` folder with one of these names:

- `W95FA.woff2` (preferred - best compression)
- `W95FA.woff` (fallback)
- `W95FA.ttf` (fallback)
- `W95FA.otf` (fallback)

The CSS is already configured to load the font automatically. The font will be used throughout the website once the file is added.

## Where to get W95FA font:

You can download W95FA font from:
- https://www.dafont.com/w95fa.font
- Or search for "W95FA font download" online

Once you add the font file to this folder, refresh your browser and the font will be applied!
