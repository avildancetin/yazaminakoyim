# Background Image Setup

To use your vintage green/yellow background image:

1. Place your background image file in this `public` folder
2. Name it `background.jpg` (or update the filename in `app/globals.css`)
3. The image will automatically be used as the website background

The CSS is already configured in `app/globals.css` to use `/background.jpg` as the background image.

If you want to use a different filename, update this line in `app/globals.css`:
```css
background-image: url('/background.jpg');
```
